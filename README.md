# The Questions:

- What is JSX, and why is it used?
- What is the difference between State and Props?
- What is the useState hook, and how does it work?
- How can you share state between components in React?
- How is event handling done in React?

## The Answers:

- JSX is a javaScript xml format. Very significant usage of it is its something where we can write html and  javaScript together which really help us save many times, makes code lesser and many more
- State is locally manageable data while props are data passed from components to components
- useState hook is a very usefool hook that helps us change, set a component's states
- Through props(a method generally known as lifting up, which basically is just sharing props)
- by capturing an onclick trigger that calls a function, we can call more click functions inside that one function, meaning a single click can run multiple operations in an website, even from one component to another
